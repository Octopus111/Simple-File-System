
# 文件管理系统 (File Management System)

## 项目描述
该项目是一个基于 Flask 的简单文件管理系统，允许用户通过 Web 界面上传、浏览、删除和重命名文件。用户可以选择上传单个文件或整个文件夹，并能够预览常见文件类型，如图片、文本和 PDF。

## 功能列表
1. **文件上传**：支持单个文件或文件夹上传，文件自动保存在 `uploads` 目录下。
2. **文件浏览**：通过 Web 界面可以查看文件和文件夹列表，支持按文件名称排序。
3. **文件删除**：可以删除单个文件或非空文件夹。
4. **文件重命名**：用户可以为文件或文件夹设置新的名称。
5. **文件搜索**：可以通过文件名搜索文件。
6. **文件预览**：支持预览图片（png, jpg, jpeg, gif）、文本文件（txt）和 PDF 文件。

## 安装与运行

### 1. 克隆项目
```bash
git clone <项目地址>
```

### 2. 安装依赖
在项目根目录下执行以下命令安装依赖：
```bash
pip install -r requirements.txt
```

### 3. 运行项目
在终端中执行以下命令启动 Flask 服务器：
```bash
python app.py
```
启动后，可以在浏览器中访问 `http://localhost:5000` 进行文件管理。

## 项目结构
```
.
├── app.py                 # 主程序文件
├── templates/             # HTML 模板文件
│   ├── browse.html        # 文件浏览和操作的页面
│   ├── preview_image.html # 图片预览页面
│   ├── preview_text.html  # 文本预览页面
│   └── preview_pdf.html   # PDF 预览页面
├── static/                # 静态文件，如 CSS、JS
└── uploads/               # 上传的文件会保存在此目录
```

## 使用方法

### 1. 上传文件或文件夹
- 点击“选择文件”按钮上传单个文件，或点击“选择文件夹”上传整个文件夹。
- 文件或文件夹上传后会显示在文件列表中。

### 2. 浏览文件
- 用户可以点击文件名来下载文件，或点击文件夹名进入子目录。
- 在文件名旁边会显示文件大小和上传时间。

### 3. 文件操作
- **重命名**：点击文件或文件夹旁的“重命名”按钮，输入新名称后点击“确认”即可完成重命名。
- **删除**：点击文件或文件夹旁的“删除”按钮即可删除该文件或文件夹。对于非空文件夹，删除操作将递归删除该文件夹及其所有内容。

### 4. 搜索文件
- 用户可以在顶部的搜索栏中输入文件名进行搜索，搜索结果会按匹配度显示在页面上。

## 注意事项
1. 请确保 `uploads` 文件夹存在。如果不存在，该系统将自动创建。
2. 该项目仅限于局域网内使用。如果需要通过外网访问，请配置 Flask 服务器绑定的 IP 或使用代理服务器。
